# Product Requirements Document (PRD)

## 삼성 광고 제너레이터 - 사이아드 (SaiAd)

**Version**: 1.0
**Last Updated**: 2025-01-21
**Status**: Draft

---

## 1. 제품 개요

### 1.1 비전
"누구나 5분 만에 프로급 삼성 제품 광고를 만들 수 있다"

### 1.2 문제 정의

| 현재 상황 | 문제점 |
|-----------|--------|
| 전문 영상 제작 | 비용 500만원+, 제작기간 2주+ |
| 템플릿 도구 (Canva 등) | 삼성 특화 X, 영상 품질 낮음 |
| AI 영상 도구 (Runway 등) | 범용적, 광고 특화 X |

### 1.3 솔루션
삼성전자 제품에 최적화된 AI 영상 생성 플랫폼
- 제품 스펙 자동 인식
- 삼성 브랜드 톤앤매너 적용
- 한국 시장 플랫폼 최적화

---

## 2. 타겟 사용자

### 2.1 Primary Persona

**셀러 민수 (32세, 쿠팡 파워셀러)**
- 삼성 가전제품 위주 판매
- 월 100개+ 상품 등록
- 영상 있는 상품 전환율 2배 경험
- 현재: 제품 사진만 사용, 영상 제작 여력 없음
- Pain Point: "영상 하나에 10만원? 마진이 안 나와요"

### 2.2 Secondary Personas

**마케터 지영 (28세, 삼성 대리점)**
- 주간 프로모션 콘텐츠 제작 담당
- SNS 채널 운영 (인스타, 틱톡)
- 빠른 트렌드 대응 필요
- Pain Point: "매번 디자인팀 요청하면 일주일 걸려요"

**인플루언서 현우 (25세, 테크 유튜버)**
- 삼성 제품 리뷰 콘텐츠 제작
- 주 2회 업로드 목표
- 인트로/아웃트로 영상 필요
- Pain Point: "편집에 시간이 너무 많이 들어요"

## 3. 핵심 기능

### 3.1 MVP 기능 (Phase 1)

#### F1. 제품 선택 시스템
```
우선순위: P0 (필수)
설명: 삼성 제품을 선택하거나 이미지 업로드

기능 상세:
- 카테고리별 제품 브라우징 (스마트폰/TV/가전/웨어러블)
- 제품 검색 (모델명, 제품명)
- 이미지 직접 업로드 (AI가 제품 인식)
- 제품 스펙 자동 로드

수용 기준:
- 주요 50개 제품 DB 구축
- 이미지 업로드 시 90% 이상 정확도로 제품 인식
- 3초 이내 제품 정보 로드
```

#### F2. 템플릿 시스템
```
우선순위: P0 (필수)
설명: 카테고리별 고품질 영상 템플릿 제공

템플릿 구성:
├── 스마트폰 (4개)
│   ├── 언박싱 시퀀스
│   ├── 카메라 하이라이트
│   ├── 일상 라이프스타일
│   └── 스펙 비교
├── TV (3개)
│   ├── 거실 시네마틱
│   ├── 게이밍 모드
│   └── 스마트홈 연동
├── 가전 (3개)
│   ├── 비스포크 인테리어
│   ├── 기능 시연
│   └── 에너지 효율
└── 웨어러블 (2개)
    ├── 헬스 트래킹
    └── 에코시스템 연동

수용 기준:
- 각 템플릿 15초/30초/60초 버전 지원
- 미리보기 3초 이내 로드
- 모바일 반응형 UI
```

#### F3. AI 스크립트 생성
```
우선순위: P0 (필수)
설명: 제품 정보 기반 광고 카피 자동 생성

톤앤매너 옵션:
- 프리미엄: 고급스럽고 세련된 톤
- 실용적: 기능과 가성비 강조
- MZ: 트렌디하고 캐주얼한 톤

생성 항목:
- 메인 헤드라인 (5자 이내)
- 서브 카피 (15자 이내)
- 나레이션 스크립트 (30초 기준 80자)
- CTA 문구

수용 기준:
- 3가지 톤별 옵션 제공
- 생성 시간 5초 이내
- 수동 편집 가능
```

#### F4. 영상 생성 파이프라인
```
우선순위: P0 (필수)
설명: AI 기반 영상 자동 생성

프로세스:
1. 템플릿 + 제품 이미지 합성
2. 텍스트 오버레이 적용
3. 전환 효과 적용
4. 배경 음악 합성
5. 최종 렌더링

수용 기준:
- 30초 영상 기준 3분 이내 생성
- 1080p 해상도 지원
- MP4 포맷 출력
```

#### F5. 편집기
```
우선순위: P1 (중요)
설명: 생성된 영상 미세 조정

기능:
- 텍스트 내용/위치/스타일 수정
- 배경 음악 변경 (5개 프리셋)
- 클립 순서 변경
- 트림/컷

수용 기준:
- 실시간 미리보기
- Undo/Redo 지원
- 자동 저장
```

#### F6. 결제 시스템
```
우선순위: P0 (필수)
설명: 크레딧 기반 과금

가격 정책:
- 무료: 월 3개 영상 (워터마크 포함)
- Basic: ₩19,900/월 - 30개 영상
- Pro: ₩49,900/월 - 100개 영상
- Enterprise: 별도 협의

수용 기준:
- 토스페이먼츠 연동
- 구독/단건 결제 지원
- 영수증 발행
```

## 5. 비기능 요구사항

### 5.1 성능
| 항목 | 목표 |
|------|------|
| 페이지 로드 | < 3초 |
| 영상 생성 (30초) | < 3분 |
| API 응답 | < 500ms |
| 동시 사용자 | 100명 |

### 5.2 확장성
- 수평 확장 가능한 아키텍처
- 영상 생성 작업 큐 시스템
- CDN 기반 영상 배포

### 5.3 보안
- OAuth 2.0 인증
- API Rate Limiting
- 업로드 파일 검증
- HTTPS 필수
